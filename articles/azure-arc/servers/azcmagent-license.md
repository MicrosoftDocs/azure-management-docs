# `azcmagent license`

Show the license agreement for the Azure Connected Machine agent.

## Usage

```
azcmagent license [flags]
```

## Examples

Show the license agreement:

```
azcmagent license
```

## Flags

This command supports the flags described in [Common flags](azcmagent.md#common-flags).
