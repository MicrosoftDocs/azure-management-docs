# `azcmagent version`

Shows the version of the currently installed agent.

## Usage

```
azcmagent version [flags]
```

## Examples

Show the agent version:

```
azcmagent version
```

## Flags

This command supports the flags described in [Common flags](azcmagent.md#common-flags).
