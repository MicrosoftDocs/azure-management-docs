# Curated resources for getting started with Azure Linux

## Videos

### Overview and introductions

**[Azure Linux: What, Why, and Where](https://youtu.be/68_7zfszkzM?si=4MxmB3cUKugTuSRh)**: John Savill's Technical Training walks through what Azure Linux is, why Microsoft built it, and where it fits in the Azure ecosystem.
